# Charts

Simple charts react app 

## Start

Install packages

```
npm install
```

Start dev server

```
npm run dev
```

## Build

```
npm run build
```

## Static code analysis

```
npm run lint
```

<code>--fix</code> flag avalilable as well:

```
npm run lint:fix
```
